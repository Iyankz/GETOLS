# GETOLS
**Gateway for Extended OLT Services**

GETOLS is an open-source gateway platform designed to simplify and unify multi-vendor OLT and ONU operations through a centralized command execution layer and observability dashboard.

This repository represents the **first public prerelease** of the GETOLS concept, architecture, and operational workflow.
